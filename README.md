# image_describer

## Installation
Install the package using pip:
```bash
pip install git+https://github.com/jzsmoreno/image_describer
```

## Usage
To run example usage, use:
```bash
python examples/example_usage.py
```

## Dependencies
- transformers
- torch
- matplotlib
- qwen_vl_utils
